# Davy

## Commandes utilisés (Terminal)
- npm init -y
- npm install express
- npm install ejs
- npm install axios
- npm install dotenv
- npm install --save-dev nodemon

## Commandes créées (Package.json)
- "scripts": {"start": "nodemon src/server.js"}

## Démarrer le site (Terminal)
````shell
npm install
npm run start
````
